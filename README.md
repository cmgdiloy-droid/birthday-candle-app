# Birthday Candle Counter App

A simple React app that allows users to add or remove birthday candles to represent age.

## Features
- Increment and decrement age
- Display candles dynamically using emojis
- Reset functionality
- Input field to set age manually
- Data persistence using localStorage

## Technologies Used
- React
- JavaScript
- CSS

## Purpose
This project was created to practice React hooks such as useState and useEffect while building an interactive and fun UI.
